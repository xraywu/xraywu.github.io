使用 ExpressVPN 和小米路由器搭建翻墙路由

發表於 2018-02-14 | 分類於 动动手 | 閱讀次數

ExpressVPN（需翻墙）是一家美国 VPN 供应商，提供同时在线 3 台设备、不限流量的高质量收费 VPN 服务。其服务器遍布全球，支持多种协议（PPTP、L2TP 或 OpenVPN）和多种设备客户端（包括移动、桌面、路由器甚至 PS 游戏机等）。简单来说，是市面上最好的收费 VPN 服务之一。当然，高质量的服务收费也很感人，一次性购买一年的费用高达 100 刀！不过，考虑这两年的国内网络环境，这样稳定的梯子实在是不可奢求更多。这不，前几天我就咬咬牙入手了一个账号，果然生活质量得到了极大的提高~

花了这么多钱买的梯子，自然想不只是简单在设备上简单使用啦（况且来回切换网络也很麻烦）。想了想，决定在家里原有路由器之外，再搭个翻墙路由，也好让买了许久，却因为网络环境吃灰许久的 Amazon Echo 能在无墙网络里用起来。看了一圈，决定入手个小米路由器3G。本以为其自带的 VPN 设置可以轻松完成任务，却发现小米路由器固件只支持简单的 L2TP 协议（都没有IPSec！）。折腾了两天，终于把它改造好了。下面做些记录，以备不时之需（你懂的）。

获取小米路由器的 Root 权限

小米路由器默认出厂刷的是稳定版，要获取 Root 权限，首先需要去其官网下载相应型号的最新开发版 ROM 并刷机。下载完 ROM 后，从浏览器访问192.168.3.1进入小米路由器默认的管理后台，在右上角菜单中选择系统升级，并选择刚刚下载的升级包进行安装。安装完成后，等待路由器重启完成。

安装完开发版系统后，需要使用小米官方提供的 Root 工具。需要注意的是，下载链接需要通过小米路由器提供的网络进入，并且需要登录小米账号。

下载 Root 工具后，不要修改文件名（miwifi.bin），并且将文件拷到 U 盘中。将 U 盘插入小米路由器，断电，然后按住 Reset 键不放再恢复电源。按住 Reset 键并开机约 10 秒后（官网说 3-5 秒，实测需要按更长一些），会进入刷机模式，放开 Reset 键，等刷机完成后重新启动。重启完成后，路由器的 SSH 登录即被激活。此时，使用如下命令 SSH 远程连接设备若成功（Root 密码在下载 Root 工具时会在小米官网上显示），证明已经获得路由器 Root 权限。

使用 OpenVPN 连接 ExpressVPN

首先，在这里，我们假设你已经购买了 ExpressVPN 的服务。正如上文所说，基于 L2TP 协议的 VPN 在目前的网络状况下几乎没有成功连接的可能性。因此，在开始配置之前，你需要访问首先访问 ExpressVPN，获取你的 OpenVPN 信息。在这个页面，你能够找到你的 OpenVPN 账号和密码、ExpressVPN 各个服务器的 OpenVPN 配置文件（.ovpn 文件）、以及访问需要的证书和公私钥。记录或下载后，进入 Padavan 后台开始配置。
